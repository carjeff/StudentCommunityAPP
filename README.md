# StudentCommunityAPP
This is a new app about student community.

## 要求

> 1. 文章列表
> 2. 文章内容显示
> 3. 评论
> 4. 关注功能
> 5. 个人中心
> 6. 美观的UI
> 7. 课程列表
> 8. 视频播放
> 9. 音频播放
> 10. 其他扩展功能
